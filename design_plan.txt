Το πρόγραμμα από την πλευρά του client θα είναι γραμμένο σε γλώσσα Python.
Θα διαβάζει ρυθμίσεις από αρχείο όπως η ip/url του server ώστε να ξέρει που να
μεταφέρει τα δεδομένα και το πόσο συχνά θα δειγματοληπτεί για αυτά. Η συλλογή των 
δεδομένων θα γίνεται κυρίως από το /proc του λειτουργικού συστήματος.Τα δεδομένα 
αυτά θα μεταφέρονται στον server με POST σε ένα REST API.

Το πρόγραμμα του server θα είναι επίσης γραμμένο σε python. Αυτό θα λαμβάνει τα 
POST requests από τους clients και θα  αποθηκεύει όλες τις πληροφορίες που δέχεται 
σε μία βάση δεδομένων. Για βάση δεδομένων θα χρησιμοποιηθεί η postgresql. 
Στον server θα υπάρχει επίσης ένα web app σε django web framework το οποίο θα 
αναλαμβάνει να διαβάζει τα δεδομένα από την βάση και να τα παρουσιάζει στον
χρήστη.

Για την επικοινωνία των 2 προγραμμάτων θα χρησιμοποιηθούν οι εξής τρόποι:
Κατά την δημιουργία του node, θα δημιουργείται ένα αρχείο με τις επιθυμητές
ρυθμίσεις για το πρόγραμμα του client. O server θα έχει ένα api στο οποίο
ο client θα μπορεί να κάνει POST requests ώστε να αποθηκευτούν στην βάση τα
δεδομένα. To authentication στο POST θα γίνεται με tokens που θα εκδίδει
ο server.

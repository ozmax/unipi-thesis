Το πρόγραμμα από την πλευρά του client θα είναι γραμμένο σε γλώσσα Python.
Θα διαβάζει ρυθμίσεις όπως η ip/url του server ώστε να ξέρει που να μεταφέρει τα
δεδομένα και το πόσο συχνά θα δειγματοληπτεί για αυτά. Η συλλογή των δεδομένων
θα γίνεται κυρίως από το /proc του λειτουργικού συστήματος.
Τα δεδομένα αυτά θα μεταφέρονται στον server με POST σε ένα REST API.

Το πρόγραμμα του server θα είναι επίσης γραμμένο σε python.
Αυτό θα λαμβάνει τα POST requests από τους clients και θα  αποθηκεύει όλες
τις πληροφορίες που δέχεται σε μία βάση δεδομένων. Για βάση δεδομένων θα 
χρησιμοποιηθεί η postgresql.

Επικοινωνία των 2 προγραμμάτων
Η επικοινωνία θα γίνεται μέσω της τεχνολογίας REST και
σύμφωνα πάνω σε ένα πρωτόκολλο που θα οριστεί.
